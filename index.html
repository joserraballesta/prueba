ACCESIBILIDAD EN LA WEB

Cualquier persona, incluyendo aquellas con discapacidades, puede usar internet. Esto implica diseñar sitios web de manera que sean fáciles de ver, entender y usar para todos, incluso para aquellos con dificultades visuales, auditivas, motoras o cognitivas.

Además, la accesibilidad web también ayuda a las personas mayores cuyas habilidades pueden disminuir con la edad.

Datos relacionados con la accesibilidad

Información sobre la accesibilidad de los sitios web
Categoría Aspecto Descripción
Visión Contraste Proporcionar suficiente contraste entre el texto y el fondo para mejorar la legibilidad para personas con discapacidad visual.
Redimensionamiento de texto Permitir que los usuarios ajusten el tamaño del texto sin que se pierda la funcionalidad o la estructura del sitio.
Audición Subtítulos Proporcionar subtítulos para contenido multimedia para personas con discapacidad auditiva.
Movilidad Teclado Asegurar que todas las funciones del sitio sean accesibles mediante el uso exclusivo del teclado para usuarios con discapacidad motora.
Compatibilidad con lectores de pantalla Garantizar que el sitio sea compatible con lectores de pantalla para usuarios con discapacidad visual.

Herramientas de analisis de Accesibilidad

WAVE Web Accessibility Evaluation Tool

WAVE es una herramienta en línea que ayuda a identificar posibles problemas de accesibilidad en páginas web.

Ir a WAVE
Color Contrast Checker de WebAIM

Esta herramienta permite verificar el contraste entre dos colores, lo que ayuda a garantizar que el texto sea legible para personas con discapacidad visual.

Ir al Color Contrast Checker
Screen Reader Online

Esta herramienta simula cómo un lector de pantalla interpreta una página web. Ayuda a las personas con discapacidad visual a convertir el texto en voz o braille.

Ir al Screen Reader Online
